# Contributing to AI Content Detector

Thanks for your interest in contributing.

## How to contribute

All changes to this repository go through pull requests. Direct pushes to
`main` are not accepted.

1. Fork the repository
2. Create a branch for your change (`git checkout -b your-feature`)
3. Make your changes
4. Open a pull request against `main` with a clear description of what you
   changed and why
5. The author will review and either approve, request changes, or decline

There is no guarantee that every PR will be merged. Changes that conflict
with the project direction or introduce unnecessary complexity may be
declined even if technically correct.

## What makes a good PR

- Focused — one logical change per PR
- Described — explain the problem being solved, not just what changed
- Clean — no unrelated formatting changes or refactors bundled in
